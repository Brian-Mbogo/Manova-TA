# Manova Travel Agency

Manova is a responsive travel agency website built with HTML, CSS, and Bootstrap.

## Live Demo

[https://manova-ta.vercel.app](https://manova-ta.vercel.app)

## Features

- Responsive navbar with mobile toggle
- Hero section with call-to-action
- Destinations cards
- Featured tours accordion
- Traveler comments/quotes section
- Contact form layout
- Social media footer links

## Project Structure

- `index.html` - Main website page
- `css/style.css` - Custom styling
- `images/` - Logo and travel images

## Run Locally

1. Clone or download the project.
2. Open `index.html` in your browser.

## Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
